# Sobeys distribution throughput

**S — Situation:** As a supply-chain intern, I noticed distribution lead times varied a lot across regions with no clear owner.
**T — Task:** Figure out where the time was going and whether it was fixable.
**A — Action:**
- Wrote SQL to join shipment, inventory, and labour tables across 3 regions.
- Isolated the slowest hand-off (a cross-dock step) as the bottleneck, not the obvious culprits.
- Built a Tableau dashboard so regional managers could see it themselves in their weekly review.
**R — Result:** Proposed a routing change projected to cut average lead time ~12%; the dashboard became a standing report.
**R — Reflection:** The win was making the data legible to non-analysts — a tool nobody uses is worthless, so I designed for the manager who didn't want to learn a new system.

# Corpus — your writing samples

Drop writing here for the agent to mine when building your voice (see `../../README.md` for the build).

What's useful, roughly best-first:
- **Past cover letters / application essays** you've written — your real voice under pressure.
- **Writing you admire** — a letter, essay, or post you wish you'd written (label it "admired" so the agent
  knows it's aspirational, not yours).
- **Sent emails / dictated notes** — long-form explaining or arguing prose is highest-signal.

One line at the top of each file: is this *yours* or *admired*? That's all the labeling the agent needs.

The agent reads these, extracts contrastive pairs into `../../pairs/`, and drafts your `../../voice-profile.md`
from what they actually show — then interviews you only for the gaps. This folder can stay messy; it's raw input.
